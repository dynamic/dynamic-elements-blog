## Requirements

- SilverStripe 4
- silverstripe/blog
- dnadesign/silverstripe-elemental

## Installation

`composer require dynamic/dynamic-elements-blog`

## Usage

dynamic-elements-blog will add the following Element to your site:

* Recent Blog Posts
